# 📊 Sales Data Analysis Dashboard

## 🎯 Objective
To analyze sales data and create an interactive dashboard using Excel.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## 📈 Features
- Sales by Region
- Profit by Product
- Sales Trend
- KPI Metrics

## 🔍 Key Insights
- South region leads in sales
- Laptop generates highest profit
- Sales show steady growth

This project demonstrates data analysis and dashboard creation skills.
